# Design brief · Margin Notes

**Aesthetic direction:** Thermal receipt printer — warm paper, orange heat accent, perforated tear lines, stacked receipt slips with slight tilt.

## frontend-design checklist

- [x] **Typography** — Fraunces (display) + Sometype Mono (receipt data) + Noto Sans KR
- [x] **Color** — parchment paper + thermal orange `#d35400`; no purple gradients
- [x] **Layout** — capture strip full-width, stats sidebar, receipt stack column
- [x] **Motion** — staggered reveal, receipt-in animation, print-button press depth
- [x] **Texture** — register grid, thermal glow, perforated edges, receipt top fade bar
- [x] **States** — empty stack, parked strikethrough, toast, focus underline
- [x] **Bilingual** — EN/KO toggle; mobile column stack at 720px

## Remember

Memorable detail: perforated tear lines + physical print button with 3D shadow press.
