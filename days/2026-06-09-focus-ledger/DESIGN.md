# Design brief · Focus Ledger

**Aesthetic direction:** Editorial ledger — warm paper stock, red margin line, numbered panels, serif headlines with mono data.

## frontend-design checklist

- [x] **Typography** — Newsreader (display) + JetBrains Mono (data) + Noto Sans KR
- [x] **Color** — parchment paper, forest focus / slate break / amber drift; red margin accent
- [x] **Layout** — asymmetric two-column with tall timer panel; numbered 01/02/03 panels
- [x] **Motion** — staggered `reveal` load, row-in ledger entries, pomodoro ring progress
- [x] **Texture** — SVG grain overlay + atmospheric radial gradients per mode
- [x] **States** — active chips, empty ledger, toast, mode-driven `body[data-mode]`
- [x] **Bilingual** — EN/KO toggle; Korean title block; no layout break at 375px

## Remember

Memorable detail: rotated privacy stamp + red margin line like a bound journal.
